# Это хорошо

Черновые заметки и концепция философской sci-fi истории.

Ключевые идеи:

- Бог создаёт мир через многоуровневую систему ангелов.
- Ангелы постепенно теряют исходный замысел.
- Бог пытается создать существо со свободой воли.
- Человек становится существом, наиболее похожим на Бога.
- Люцифер — самый совершенный ангел, не способный принять свободу.
- «Это хорошо» означает не «идеально», а «подлинно и свободно».

Планируется:

- философская повесть;
- радиодрама / почти пьеса;
- серия диалоговых сцен и montage failed universes.
